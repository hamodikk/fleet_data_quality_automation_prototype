# Data Sources

This document tracks the public datasets used as structural inspiration for the mock fleet datasets. These sources informed column ideas, realistic data-quality issues, and business context, but the final project uses generated mock data rather than real operational records from these sources.

| Data Name | Date Downloaded | Description | Data Source | Link |
|---|---|---|---|---|
| Fleet_Preventative_Maintenance_&_Repair_Work_Orders_20260524 | 05/24/2026 | This dataset contains all information on work orders completed for assets from January 2008 to present. | City of Cincinnati | https://data.cincinnati-oh.gov/Thriving-Neighborhoods/Fleet-Preventative-Maintenance-Repair-Work-Orders/2a8x-bxjm/about_data |
| vehicle_maintenance_data | 05/24/2026 | This dataset provides synthetic data related to vehicle maintenance to help predict whether a vehicle requires maintenance or not based on various features. | Chavindu Dulaj, Kaggle | https://www.kaggle.com/datasets/chavindudulaj/vehicle-maintenance-data?resource=download |
| Copy of Fuel Usage 18-19 | 05/24/2026 | Dataset showing how much fuel each council vehicle has consumed, by year and by fuel type. | Leeds City Council, Data Mill North | https://datamillnorth.org/dataset/fleet-vehicles-fuel-consumption-20jl1 |
