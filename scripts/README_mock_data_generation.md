# Mock Data Generation Notes

The mock data generation script was created with AI assistance and reviewed for alignment with the project goals.

The generated datasets are synthetic and were designed to support testing of data profiling, exception generation, cleanup, review workflows, and dashboarding. Public fleet related datasets were used as structural inspiration, but the final datasets are populated with mock data.
